Gemeinsam für ein gutes Klima

Ein Beispiel-Tool, das das Grüne AT Design System einbindet.

Diese Seite verwendet ausschließlich die zentral gepflegten Komponenten des Designsystems. Wer ein neues Grüne-AT-Tool baut, kann diese Datei als Ausgangspunkt nehmen: das gehostete Stylesheet einbinden, das body-Grundlayout setzen und die Oberfläche aus den `gat-`-Klassen zusammensetzen.

Worum es geht

Eine Karte fasst eine einzelne Aussage kompakt zusammen. Die Primär-Variante steht auf dunkelgrüner Fläche mit weißem Text.

Wie es weitergeht

Die Sekundär-Variante steht auf hellgrüner Fläche mit Anthrazit-Text. Für jede Karte ist ein Varianten-Modifier Pflicht.

Hinweis: Dies ist ein Warnungs-Callout aus v2.1 — eine von fünf semantischen Varianten (info, warn, error, success, legal).

Wir melden uns innerhalb von 48 Stunden.
