03Decisions

Key decisions & options

D-1 — Bezug zum persönlichen System entfernen

Gesetzt

Alle Verweise auf flomotlik.github.io/claude-code werden aus Produktdateien entfernt. Inventur: nur README.md Z. 22–33 betroffen.

Querverlinkung persönliches ↔ org-System (ursprüngliche ISSUE.md-Idee) — verworfen.

Treffer unter .issues/ mitlöschen — verboten; das sind Tooling-Artefakte.

D-2 — Versionierung: Latest-URL + CHANGELOG

Gesetzt

Feste Pages-URL; CHANGELOG.md im Keep-a-Changelog-1.1.0-Format mit SemVer 2.0.0. Breaking Changes als MAJOR-Sprung angekündigt.

Versionierte URLs / CalVer — verworfen; kommunizieren keine Breaking Changes.

D-3 — Minimal-Beispiel als echte Datei

Gesetzt

examples/minimal.html bindet ausschließlich die absolute, gehostete design-system.css-URL ein und nutzt echte gat--Komponenten.

Relative CSS-Einbindung — verworfen; würde das Hosting nicht dogfooden.
